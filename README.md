# Hey Soulmate Website

Award-style single-page website for Hey Soulmate, a Hong Kong mindful speed dating brand for 28-39 young professionals.

## What is included

- Immersive editorial hero
- Scroll reveal animation
- Progress indicator
- Parallax image treatment
- Responsive mobile layout
- Past event highlight for the 100-person masked date at Eaton HK

## Preview locally

Open `index.html` directly, or run a local static server from this folder:

```bash
python3 -m http.server 5178 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:5178/
```
